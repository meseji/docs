Programmatic access to read and write data in Meseji

Meseji's API can be used to programmatically access and take action on data within Meseji.

Only Admin users can create API tokens. To learn more about Admins, see [Roles & User Management](https://docs.usepylon.com/pylon-docs/platform/roles-and-user-management).

Some common use cases for our API include:

- Fetching information about customers
- Manage Projects and Role
- Fetching information about Campgains
- Creating Template and Manage
- Manage Flow of User Query And Conversations

Check out our [API Reference](https://docs.usepylon.com/pylon-docs/developer/api/api-reference) for specifications on all endpoints.
