<Note>
  Before sending a template message, use this endpoint to understand what variables are required.
</Note>

## Overview

This endpoint returns the dynamic variables (placeholders) that a specific template requires. Variables can include text fields, image URLs, document links, and button parameters.

## Path Parameters

| Parameter | Type   | Required | Description                                                 |
| --------- | ------ | -------- | ----------------------------------------------------------- |
| `name`    | string | Yes      | The name of the WhatsApp template (e.g., `welcome_message`) |

## Variable Types

<AccordionGroup>
  <Accordion title="TEXT" icon="font">
    Plain text variables used in the message body or header.

    ```json
    {
      "label": "Customer Name",
      "keyName": "body_1",
      "type": "TEXT"
    }
    ```
  </Accordion>
  <Accordion title="IMAGE_URL" icon="image">
    URL pointing to an image file for header media.

    ```json
    {
      "label": "Header Image",
      "keyName": "header_image",
      "type": "IMAGE_URL"
    }
    ```

    **Supported formats:** JPG, PNG\
    **Max size:** 5MB
  </Accordion>
  <Accordion title="DOCUMENT_URL" icon="file">
    URL pointing to a document file for header media.

    ```json
    {
      "label": "Invoice Document",
      "keyName": "header_document",
      "type": "DOCUMENT_URL"
    }
    ```

    **Supported formats:** PDF\
    **Max size:** 100MB
  </Accordion>
  <Accordion title="VIDEO_URL" icon="video">
    URL pointing to a video file for header media.

    ```json
    {
      "label": "Promo Video",
      "keyName": "header_video",
      "type": "VIDEO_URL"
    }
    ```

    **Supported formats:** MP4\
    **Max size:** 16MB
  </Accordion>
</AccordionGroup>

## Example Response

```json
{
  "status": true,
  "message": "Template variables fetched successfully",
  "data": [
    {
      "label": "Header Image",
      "keyName": "header_image",
      "type": "IMAGE_URL"
    },
    {
      "label": "Customer Name",
      "keyName": "body_1",
      "type": "TEXT"
    },
    {
      "label": "Order ID",
      "keyName": "body_2",
      "type": "TEXT"
    },
    {
      "label": "Track Order Link",
      "keyName": "button_1",
      "type": "TEXT"
    }
  ]
}
```

## Using Variables

Once you have the variables, use them when sending a template message:

```json
{
  "to": "919876543210",
  "from": "919123456789",
  "language": "en_US",
  "template_id": "order_confirmation",
  "variables": [
    {
      "keyName": "header_image",
      "value": "https://example.com/order-banner.jpg"
    },
    {
      "keyName": "body_1",
      "value": "John Doe"
    },
    {
      "keyName": "body_2",
      "value": "ORD-12345"
    },
    {
      "keyName": "button_1",
      "value": "https://example.com/track/ORD-12345"
    }
  ]
}
```

<Warning>
  All required variables must be provided when sending a template message. Missing variables will cause the message to fail.
</Warning>
