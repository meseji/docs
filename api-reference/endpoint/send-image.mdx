<Note>
  Image messages can only be sent within the 24-hour messaging window after a customer initiates contact.
</Note>

## Overview

Send image messages to your customers via WhatsApp. Perfect for sharing product images, receipts, QR codes, infographics, and visual content that enhances your communication.

## Request Body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `to` | string | Yes | Recipient's phone number with country code |
| `from` | string | Yes | Your WhatsApp Business phone number |
| `image_url` | string | Yes | Public URL of the image to send |
| `caption` | string | No | Optional text caption (max 1024 characters) |

## Image Requirements

<CardGroup cols={2}>
  <Card title="Supported Formats" icon="file-image">
    - JPEG / JPG
    - PNG
  </Card>
  <Card title="Size Limits" icon="weight-scale">
    - Maximum file size: **5 MB**
    - Recommended: Under 1 MB for faster delivery
  </Card>
</CardGroup>

<Warning>
  The image URL must be publicly accessible. Private URLs, localhost, or URLs requiring authentication will not work.
</Warning>

## Best Practices

<Steps>
  <Step title="Optimize Image Size">
    Compress images before uploading to ensure fast delivery. Tools like TinyPNG can help reduce file size without quality loss.
  </Step>
  <Step title="Use CDN URLs">
    Host images on a CDN (CloudFront, Cloudflare, etc.) for reliable and fast access worldwide.
  </Step>
  <Step title="Add Meaningful Captions">
    Captions provide context and can include calls-to-action, making your images more effective.
  </Step>
  <Step title="Test URLs First">
    Verify that image URLs are accessible by opening them in a browser before sending.
  </Step>
</Steps>

<Tip>
  Caption text supports the same formatting as text messages: `*bold*`, `_italic_`, `~strikethrough~`, and `` `monospace` ``.
</Tip>
